# 安装要求
本项目需要下载安装后在DevEco Studio上运行
# 安装步骤
## 1.软件安装
安装Windows环境的DevEco Studio 3.0.0.800并在huawei device manager中安装本地的phone模拟器
## 2.配置开发环境
配置proxy，安装Node.js和ohpm并下载java的HarmonyOS SDK
